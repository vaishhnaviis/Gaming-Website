ONLINE GAMING
A NEW WEBSITE FOR GAMERS

This project is aimed at developing online games. The website provides multiple games which are very popular and known to huge crowd. The projects will be majorly based on HTML, CSS, JavaScript to build the games. Aim of the project is to provide games of all genre such as strategy games, adventure games, puzzle games, guessing games, board games etc. Few games include Stone-paper-Scissors, Angry bird game, racing game, Tic-tac-toe, Shooting game etc. The main highlight of the project is to provide multiple games at a single platform without any registration. This makes the website unique from other website. We believe that the project will be a great success as it provides an easy and convenient platform for people to access multiple online games and will be able to reach a large audience.
